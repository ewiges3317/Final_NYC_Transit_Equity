🚇 NYC Transit Equity Analysis

A Data-Driven Call for Action in Late-Night Public Transportation

📊 Executive Summary

Uncovering critical disparities in NYC's overnight transit system

59% Lower East Side Overnight Dependency
2.3M+ Overnight Trips Analyzed
265+ NYC Taxi Zones Studied
70% Highest Zone Dependency (Williamsburg North)

🎯 Key Findings

Geographic Disparities: Significant overnight transit reliance varies dramatically across NYC boroughs and zones

Service Gaps: Critical gaps exist during 10PM-1AM when demand peaks in key zones

Affected Communities: Late-night workers and lower-income neighborhoods face greatest burden

🔬 Research Methodology

📈 Data Source

  • Dataset: NYC Taxi & Limousine Commission Yellow Taxi data (2024)
  • Coverage: February, April, July, October 2024
  • Sample Size: 13.4M raw trips → 13.2M cleaned trips
  • Overnight Focus: 2.3+ million overnight trips (10PM-5:59AM)
  • Geographic Scope: All 5 boroughs + Newark Airport
  • Analysis Zones: 265+ taxi zones (min. 10K trips per zone)

💻 Technical Stack

Python DuckDB pandas matplotlib seaborn Jupyter SQL

📊 Key Metric Definition

Overnight Reliance Formula:

(Overnight Trips ÷ Total Trips) × 100

Percentage of total trips occurring during overnight hours (10:00 PM to 5:59 AM)

🏗️ Technical Implementation

🔄 19-Step SQL Workflow

  • Data Processing (Steps 1-4): Union monthly Parquet files, clean data, add overnight flags, enrich with zone lookups
  • Analysis (Steps 5-9): Calculate overnight reliance by zone and borough, identify peak hours and patterns
  • Validation (Steps 10-13): Row totals, fare sanity checks, tip analysis for data quality assurance
  • Export (Steps 14-19): Generate CSV outputs for visualization and further analysis
